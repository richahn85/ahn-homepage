# 안영찬 홈페이지 초기 콘텐츠안

## 1. 홈페이지의 첫 메시지

**핵심 포지션**

안영찬 교수의 홈페이지는 단순 프로필 페이지가 아니라, AX(AI 전환) 시대의 경영 의사결정을 돕는 리포트 구독, 특강, 자문, 컨설팅 접점으로 운영한다.

**첫 화면 문구**

30년의 경영 컨설팅 위에 AI를 얹어, 실행되는 전략을 매달 당신의 책상 위에.

**보조 설명**

현대백화점 국내 최초 CRM, 국내 최초 K-BPI 브랜드파워 지수, 국내 최초 중대재해처벌법 대응 프로그램 등 "국내 최초"를 만들어 온 경험을 바탕으로, AX(AI 전환) 시대의 다음 수를 리포트와 자문으로 전한다.

**대표 CTA**

- 월간 리포트 구독하기
- 자문·컨설팅 문의

## 2. 초기 홈페이지 구성

1. **Hero**
   - 안영찬 교수의 정체성과 핵심 메시지 제시
   - 30년+ 경험, 국내 최초 프로젝트, 강의·자문 기업, 발행 보고서 수를 신뢰 지표로 배치

2. **월간 AX 경영 브리프 구독**
   - 홈페이지의 1차 수익 모델
   - 매월 첫째 주 발행, PDF 12~16면
   - 이달의 판단, 현장 프레임, 실행 체크리스트, Q&A 코너로 구성
   - 개인 구독, 팀·기업 구독, 기업 구독 + 자문 결합으로 플랜 분리

3. **서비스와 비용**
   - 월간 AX 경영 브리프 구독
   - AI·AX 전략 특강과 워크숍
   - 경영진 정기 자문
   - AX 실행 로드맵 컨설팅

4. **전문분야**
   - AX(AI 전환) 경영 컨설팅
   - AI Agentic 업무자동화
   - B2B 마케팅 전략
   - 고객경험·서비스 품질
   - 표준화·공공 컨설팅
   - 조직문화·GWP·안전경영

5. **신뢰 근거**
   - 국내 최초 프로젝트 기록
   - 삼성전자, SK Telecom, LG화학, 현대자동차, 금융권, 공공기관 등 주요 고객·기관
   - 30년+ 경영 컨설팅·교육 경험
   - 국가기술표준원 선정 국제표준 전문가 이력

6. **경력**

   - 주요 보고서·저서 목록

7. **사회공헌**
   - 사회적기업·협동조합 무료 자문
   - 소상공인 AI 활용 무료 특강
   - 비영리단체·복지기관 리포트 무료 구독

   - 사회공헌 무료 서비스

## 3. 초기 공개 버전의 운영 방향

초기 홈페이지는 "교수 소개"보다 "문의와 구독이 발생하는 공식 채널"에 초점을 둔다. 첫 화면에서 바로 구독과 문의 버튼을 노출하고, 중간에는 가격 기준을 공개해 잠재 고객이 문의 전에 대략적인 규모를 판단할 수 있게 한다.

현재 콘텐츠는 다음 순서로 공개하는 것이 적합하다.

1. `ahnyoungchan.kr` 대표 도메인
2. 구독 신청 폼과 문의 폼
3. 리포트 구독 가격과 자문·컨설팅 기준가
4. 신뢰 근거와 경력
5. 사회공헌 섹션

## 4. 배포 전 확인 사항

- Netlify 계정 크레딧 문제 해결 후 최신 파일 재배포
- Netlify Forms에서 `subscribe`, `contact` 폼 감지 확인
- Netlify Notifications에서 접수 이메일 알림 설정
- HTTPS 인증서 발급 완료 확인
- 실제 구독료와 결제 방식 최종 확인
- 후원 계좌 공개 여부 최종 확인
